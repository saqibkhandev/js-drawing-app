# JS Drawing App ✍️

A simple drawing app using the HTML5 canvas, built with HTML, CSS, and JavaScript. Great for learning canvas APIs and event handling.

## Features
- Draw with mouse/touch input
- Clear canvas / color controls (if implemented)

## Technologies Used
- HTML
- CSS
- JavaScript (Canvas)

## Project Structure
```
index.html — demo UI
styles.css — styling
script.js — drawing logic
```

## Installation / How to run
1. Clone the repo.
2. Open `index.html` in a browser.
